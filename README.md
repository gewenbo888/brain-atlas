# Brain Atlas — 人脑完全图谱

[![Live Demo](https://img.shields.io/badge/Live-brain--atlas--ten.vercel.app-e8a0c8)](https://brain-atlas-ten.vercel.app)

A comprehensive bilingual (Chinese–English) guide to the human brain — from the genes that build it to the consciousness that emerges from it.

## Sections

1. **Genes** — BDNF, COMT, APOE, 5-HTTLPR, FOXP2, MAOA and their roles in brain function
2. **Neurons** — Pyramidal, interneurons, mirror neurons, glial cells, place/grid cells, Von Economo neurons
3. **Neurotransmitters** — 9 transmitters with too-little/too-much effects and related drugs (dopamine, serotonin, GABA, glutamate, acetylcholine, endorphins, oxytocin, norepinephrine, anandamide)
4. **Brain Regions** — Prefrontal cortex, hippocampus, amygdala, cerebellum, language areas, default mode network
5. **Mental Disorders** — Depression, anxiety, schizophrenia, Alzheimer's, ADHD, autism, Parkinson's, PTSD with prevalence, mechanism, and treatment
6. **Psychology** — Cognitive biases, Maslow, attachment theory, flow state, neuroplasticity, Big Five
7. **Brain-Computer Interface** — EEG, Neuralink, Utah Array, Stentrode, optogenetics, future vision
8. **Consciousness** — IIT, Global Workspace Theory, free will, psychedelics, AI consciousness, the Hard Problem

## Design

Deep violet-black aesthetic with animated neural network hero canvas (synaptic pulses), 7-color section coding, Lora + Manrope + Noto Sans SC typography, sticky section navigation.

## Tech

Single-page HTML/CSS/JS — no build step, no dependencies.
